
# Features

## MVP Features

### 1. Natural Language Search

The broker writes the client's request in plain language.

Example:

"I need a 180 sqm apartment in New Cairo under EGP 8 million."

B.A understands the request automatically.

---

### 2. Smart Property Matching

The AI searches all available properties and ranks the best matches.

---

### 3. Advanced Filters

Search by:
- Location
- Budget
- Property Type
- Bedrooms
- Area
- Developer
- Delivery Date

---

### 4. Property Details

Each result includes:
- Images
- Price
- Area
- Floor
- Bedrooms
- Bathrooms
- Compound
- Developer
- Commission

---

### 5. Broker Notes

Brokers can add private notes for every property.

---

### 6. Favorites

Save properties for later.

---

### 7. Share Property

Share property via:
- WhatsApp
- PDF
- Link

---
